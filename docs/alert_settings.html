設定項目の説明

ticket_count_warning 買い目点数が何点以上なら「注意」を出すかの設定です。
ticket_count_danger 買い目点数が何点以上なら「強い注意」を出すかの設定です。
total_amount_warning 予定投資額が何円以上なら「注意」を出すかの設定です。
total_amount_danger 予定投資額が何円以上なら「強い注意」を出すかの設定です。
updated_hours_warning 予想更新から何時間以上経過したら「注意」を出すかの設定です。
updated_hours_danger 予想更新から何時間以上経過したら「強い注意」を出すかの設定です。
check_... 各チェック項目を有効にするか無効にするかの設定です。true でON、false でOFFです。

使い方

アラート条件を変更したい場合は、docs/alert_settings.json を編集してください。 例：買い目点数を10点以上で注意にしたい場合は、ticket_count_warning を 10 に変更します。

変更後は GitHub Pages の反映に少し時間がかかる場合があります。古い表示が出る場合は、URL末尾に ?v=32 のような文字を付けて再読み込みしてください。
